  • Functions


  • Searching -The SEARCH function is design to find or to look up the content inside the grid-view that is categorized by Student and LRN#.(base on the top image)
      • the yellow box indicate as a search function to find specific records.
      • The green box indicate is also a search functionality but it depends on what type of search is selected where it is a student name or LRN#.By default it is labeled  as "All" meaning to display the whole record.
      • Blue box is an icon that represents as a reset button.To make everything back to its normal function or default settings.  



  • Editing Student Record (refer top image)
      • The pink box is an edit function where you can change specific record.


  • Deleting Student Record (refer top image)
      • Orange box refer as a delete button inside the grid view. Once you click this it will prompt you a message box stated " Are you sure you want to delete this student?",you have to choose Yes or No.
  • Adding New Student (refer top image)
      • Gray  box is a function that will  allow the user to add new student.
