Batch Receivable

The batch receivable module, which can be access through Account Card > Batch Receivable, is where you can record student payables by batch for the following categories:

  • Catering
  • Charity
  • Extra-Curricular
  • Christmas
  • Family Day
  • Picture
  • Graduation Fee
  • Scouting/Camping
  • Nutrition Day
  • Moving Up/Recognition
  • Others



Recording a payable

First select the payment category from the dropdown list.


Then select the school year and enter the date. By default these will be already set to the current school year and the current date respectively. Then enter the receivable amount and enter the remarks.


Then below is the list of students, filtered according to grade level. You can change this filter by selecting a grade level from the dropdown.



Then there are 2 ways to select students, if you only want to select certain students, tick the checkboxs on the left of the selected students and click the first button. This will add the selected students to the list on the right



But if you want to add an entire grade level to the list on the right, select the grade level and click the 2nd button.


To remove a student from the right, either click the remove button on the row or tick the checkbox and click the remove students button.


Once you've double checked everything to be correct, click the save button above.



Edit/Delete a batch payable

To edit a previous payable record, go to the History tab and click the edit button of the selected row.


Make the necessary changes and click the save button.


To delete a payable, click the delete button instead.
